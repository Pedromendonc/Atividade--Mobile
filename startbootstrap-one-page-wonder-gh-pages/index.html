O que eu preciso saber sobre React?

by: Pedro

Leia

React JS é uma biblioteca JavaScript para a criação de interfaces de usuário — ou UI (user interface). Criado em 2011 pelo time do Facebook, o React surgiu com o objetivo de otimizar a atualização e a sincronização de atividades simultâneas no feed de notícias da rede social, entre eles chat, status, listagem de contatos e outros..

Primeiro Passo: Components

Os componentes são bits de código independentes e reutilizáveis. Eles servem ao mesmo propósito que as funções JavaScript, mas trabalham isoladamente e retornam HTML. Os componentes vêm em dois tipos, componentes de classe e componentes de função, neste tutorial vamos nos concentrar nos componentes de função..

Segundo Passo: Props

São informações que podem ser passadas que podem ser passas de um componente pai para um componente filho.

Terceiro Passo: UseEffect

O useEffect é um Hook que permite executar efeitos colaterais em seus componentes. Alguns desses efeitos colaterais podem ser: busca de dados em uma API, atualizar diretamente o VirtualDOM e controlar temporizadores (setTimeout e setInterval). O useEffect é utilizado para fazer o ciclo de vida de um componente em substituição às funções componentWillMount, componentDidMount, componentDidUpdate e componentWillUnmount.
